## Notes on notation in text

- **Base subscripts:** Always included when the base is not 10. Examples: $34_6$, $1011_2$, $4B7_{14}$.
- **Expanded form:** Uses $\times$ for multiplication and $+$ for addition, with exponents for place values. Example: $738 = 7 \times 10^2 + 3 \times 10^1 + 8 \times 10^0$.
- **Division notation for base conversion:** Uses $\div$ and "r" for remainder (e.g., $298 \div 6 = 49 \text{ r } 4$). This matches standard arithmetic notation students see in school.
